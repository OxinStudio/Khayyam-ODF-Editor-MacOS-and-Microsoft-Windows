# Khayyam PDF Editor

A free, privacy-first PDF editor with no subscriptions and no cloud uploads.
Available for **macOS** (native Swift/SwiftUI) and **Windows** (Python/PyQt6).
Developed by [Oxin Studio](https://github.com/OxinStudio).

## Features

- **Annotate** — Highlight, underline, and strikethrough text
- **Draw** — Freehand ink, rectangles, ovals, and lines (Shift to constrain to square / 45°)
- **Comment** — Sticky notes and typewriter text
- **Edit existing text** — Click any text block to modify PDF content in place (powered by MuPDF)
- **Insert images** — Place images anywhere on the page
- **Organize** — Merge multiple PDFs, split by page range or intervals
- **Navigate** — Thumbnail sidebar, outline/bookmarks, full-text search
- **Save** — Incremental or full save; drag-and-drop to open

## macOS Edition

Built with Swift, SwiftUI, PDFKit, and MuPDF.

### Requirements

| | Minimum |
|---|---|
| macOS | 13.0 Ventura |
| Architecture | Apple Silicon (arm64) |
| Xcode | 15 or later |

> Intel Macs are not supported because MuPDF is built as an arm64-only static library. To add Intel support, build MuPDF from source as a universal binary.

### Getting started

**1. Clone the repository**

```bash
git clone https://github.com/OxinStudio/khayyam-pdf-editor.git
cd khayyam-pdf-editor
```

**2. Run setup**

```bash
./setup.sh
```

This script will:
- Install [XcodeGen](https://github.com/yonaskolb/XcodeGen) via Homebrew (if not already installed)
- Check for MuPDF (install it or detect a source build)
- Generate `Config/LocalPaths.xcconfig` with the correct paths for your machine
- Generate `KhayyamPDFEditor.xcodeproj` via XcodeGen
- Open the project in Xcode

**3. Build in Xcode**

1. Select your **Team** in **Signing & Capabilities**
2. Press **Cmd+B** to build, **Cmd+R** to run

### MuPDF dependency

Homebrew ships MuPDF as a dynamic library, which is rejected by the macOS hardened runtime. For distribution you must build MuPDF from source as a static library:

```bash
curl -L https://mupdf.com/downloads/archive/mupdf-1.27.2-source.tar.gz -o /tmp/mupdf.tar.gz
tar -xf /tmp/mupdf.tar.gz -C /tmp
cd /tmp/mupdf-1.27.2-source
make HAVE_X11=no HAVE_GLUT=no
sudo make install HAVE_X11=no HAVE_GLUT=no
```

After installing, re-run `./setup.sh` — it will automatically detect the static libraries.

### Project structure (macOS)

```
Sources/
├── App/              # App entry point (@main)
├── Models/           # PDFDocument, annotations
├── ViewModels/       # PDFEditorViewModel — central state and business logic
├── Views/            # SwiftUI views (ContentView, PDFViewerView, SidebarView...)
├── MuPDF/            # Objective-C++ bridge to MuPDF C library
└── Assets.xcassets/  # App icon, accent color
Config/
└── LocalPaths.xcconfig   # (gitignored) generated by setup.sh
project.yml               # XcodeGen project spec
setup.sh                  # One-command project setup
```

### Architecture

The app follows MVVM:

- **PDFEditorViewModel** — central state and business logic, observed by all views
- **MuPDFBridge** (Objective-C++) — thin wrapper around the MuPDF C API
- **PDFTextEditEngine** — Swift model for text block extraction and editing
- **SwiftUI views** — PDFViewerView hosts an NSViewRepresentable wrapping Apple's PDFView

### Distribution

1. Set your **Team** to your **Developer ID** certificate
2. **Product > Archive**
3. Organizer: **Distribute App > Direct Distribution > Export**

## Windows Edition

Built with Python 3.10+, PyQt6, and PyMuPDF. Runs on Windows 10/11 (64-bit).

### Quick start (run from source)

```powershell
cd Windows
pip install -r requirements.txt
python main.py
```

### Build a standalone installer

**Prerequisites — install once**

```powershell
pip install pyinstaller Pillow
# Download and install Inno Setup 6: https://jrsoftware.org/isdl.php
```

**Build everything**

```powershell
cd Windows
.\build.bat
```

This produces:
- `dist\Khayyam PDF Editor\Khayyam PDF Editor.exe` — portable app bundle
- `installer\Khayyam-PDF-Editor-Setup-1.0.0.exe` — full Windows installer with Start Menu, uninstaller, and optional PDF association

Or run steps manually:

```powershell
python make_icon.py
python -m PyInstaller khayyam.spec --noconfirm
& "C:\Program Files (x86)\Inno Setup 6\ISCC.exe" installer.iss
```

### Windows file structure

```
Windows/
├── main.py              # Entry point
├── requirements.txt     # Python dependencies (PyMuPDF, PyQt6)
├── khayyam.spec         # PyInstaller spec (one-folder bundle)
├── installer.iss        # Inno Setup script -> .exe installer

├── make_icon.py         # Converts Mac PNG icons -> icon.ico
├── icon.ico             # Pre-generated Windows icon
└── src/

    ├── pdf_document.py  # Merge / split operations
    ├── toolbar.py       # Grouped annotation toolbar
    ├── sidebar.py       # Thumbnails, outline, search panel
    ├── main_window.py   # Main window, menus, nav toolbar
    └── dialogs.py       # Merge, Split, AddText, StickyNote, GoToPage dialogs
```
